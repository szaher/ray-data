## map — Row-by-Row

The `map` method applies a function to each row. Each row is a Python dictionary.

<CodeBlock language="python" code={`import ray

ds = ray.data.from_items([
    {"text": "Hello World", "label": 1},
    {"text": "Ray Data", "label": 0},
])

# Add a new field to each row
def add_length(row):
    row["text_length"] = len(row["text"])
    return row

ds = ds.map(add_length)
ds.show()
# {'text': 'Hello World', 'label': 1, 'text_length': 11}
# {'text': 'Ray Data', 'label': 0, 'text_length': 8}`} />

## map_batches — Vectorized Processing

`map_batches` is the workhorse. Your function receives a pandas DataFrame (or Arrow table) and should return one. This enables fast vectorized operations.

<CodeBlock language="python" code={`import pandas as pd

ds = ray.data.read_csv("s3://bucket/users.csv")

def normalize_ages(batch: pd.DataFrame) -> pd.DataFrame:
    batch["age_normalized"] = (batch["age"] - batch["age"].mean()) / batch["age"].std()
    return batch

ds = ds.map_batches(normalize_ages, batch_format="pandas")

# Use a class for stateful transforms (e.g., loading a model)
class Predictor:
    def __init__(self):
        import torch
        self.model = torch.load("model.pt")

    def __call__(self, batch: pd.DataFrame) -> pd.DataFrame:
        batch["pred"] = self.model.predict(batch["features"].tolist())
        return batch

ds = ds.map_batches(Predictor, concurrency=4, num_gpus=1)`} />

## flat_map — One-to-Many

Use `flat_map` when one input row should produce multiple output rows.

<CodeBlock language="python" code={`ds = ray.data.from_items([
    {"sentence": "Hello world. How are you."},
    {"sentence": "Ray Data is great."},
])

def split_sentences(row):
    return [{"word": w} for w in row["sentence"].split()]

ds = ds.flat_map(split_sentences)
ds.show(4)
# {'word': 'Hello'}
# {'word': 'world.'}
# {'word': 'How'}
# {'word': 'are'}`} />

## filter — Keep Matching Rows

<CodeBlock language="python" code={`ds = ray.data.from_items([
    {"name": "Alice", "score": 85},
    {"name": "Bob", "score": 42},
    {"name": "Carol", "score": 91},
])

ds = ds.filter(lambda row: row["score"] >= 50)
ds.show()
# {'name': 'Alice', 'score': 85}
# {'name': 'Carol', 'score': 91}`} />

## add_column — Quick Column Addition

A shortcut for adding a computed column without writing a full map function.

<CodeBlock language="python" code={`ds = ray.data.from_items([{"price": 100}, {"price": 200}])
ds = ds.add_column("price_with_tax", lambda row: row["price"] * 1.1)
ds.show()
# {'price': 100, 'price_with_tax': 110.0}
# {'price': 200, 'price_with_tax': 220.0}`} />

## Choosing the Right Method

| Method | Input | Output | Best For |
|--------|-------|--------|----------|
| `map` | 1 row (dict) | 1 row | Simple per-row transforms |
| `map_batches` | batch (DataFrame) | batch | Vectorized ops, ML inference |
| `flat_map` | 1 row | 0+ rows | Splitting, exploding |
| `filter` | 1 row | bool | Dropping unwanted rows |
| `add_column` | 1 row | 1 value | Adding a single column |
