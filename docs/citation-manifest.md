## Maintenance

When updating the course to a new Ray release:

1. Check each URL above still resolves and describes the same behavior
2. Update the "Ray Version" column to the new pinned version
3. Update the "Accessed" date
4. Add or update permalink columns with the new release tag
5. Review lesson text for any behavioral changes between versions
